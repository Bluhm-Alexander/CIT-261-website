Notes on Local Storage API

There are two different Storage Types available to us in the localStorage API. The first one i the local storage object. Local storage objects save information to the clients computer and that information will be stored without a experation date. The only thing that can expire said information is when the client clears the browser's cache.You can also clear a local storage object with localStorage.removeItem("item");.
