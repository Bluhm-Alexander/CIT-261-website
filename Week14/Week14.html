Notes: CSS3 Animations Without Libraries

Let's start with what transformations are. Transformations are the modification of CSS3 properties when an event has been triggered. Specifically transformations are to do with position and orientation, for example you can move a div box to another location within the webpage with a transform command.

You can also manipulate elements in a 3D space and skew the perspective. You can use the transition property in CSS to smoothly accomplish these transformation which will make the effect more pleasing to the user.

CSS3 is a very powerful style language which can render pretty much any simple object that you can think of.
