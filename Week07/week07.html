Notes on XMLHTTPRequests

I've decided to include notes inside my web pages now because why not. JSON web services are Web services or client side code that uses JSON to fill in information from the server. For example I am using JavaScript to read and display the Contents of a JSON file. This stuff is very useful for lets say displaying a catalog.

XMLHTTPRequests is basically a way for the web server to serve information to the user. To be more particular it is a API (Application program interface) that is integrated into javascript. XMLHTTPRequest is an object that can be allocated and then passed information in our particular case a JSON File. If the pass in was succeful and informatio is served properly we should get a 200 status.
